ScholarPress Backend — Rust monorepo for document extraction, validation, typst compilation, and CLI.

## Crates

- `sp-extract` — PDF and DOCX document extraction
- `sp-validate` — Institution formatting validation (36 checkers)
- `sp-typst` — Native Typst compilation

## Apps

- `publish-service` — Axum web server for document processing
- `scholarpress-cli` — Unified command-line interface
